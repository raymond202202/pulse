# Pulse 设置页面规划

## 问题
当前没有统一的「设置」入口。齿轮图标(⚙️)在底部环境选择器旁，只管理环境变量。
需要规范化：齿轮图标应作为全局设置入口。

## 方案
在标题栏右侧增加一个齿轮图标，作为全局设置入口。
点击后弹出设置模态框，包含多个 Tab：

### Tab 1: 通用
| 设置项 | 类型 | 说明 |
|--------|------|------|
| 语言 | 下拉 | 中文 / English（为多语言切换准备） |
| 主题 | 下拉 | 浅色 / 深色 / 跟随系统 |
| 请求超时(秒) | 数字输入 | 默认 30，范围 1-300 |
| 历史记录上限 | 数字输入 | 默认 200，范围 10-1000 |

### Tab 2: AI
| 设置项 | 类型 | 说明 |
|--------|------|------|
| API Provider | 下拉 | OpenAI / DeepSeek / 自定义 |
| API Endpoint | 文本输入 | 自定义时需填完整 URL |
| API Key | 密码输入 | 存 localStorage（非安全存储） |
| 模型 | 下拉/文本 | 如 gpt-4o / deepseek-chat / 自定义 |
| 上下文关联 | 开关 | 是否自动注入当前请求/响应 |

### Tab 3: 代理
| 设置项 | 类型 | 说明 |
|--------|------|------|
| HTTP 代理 | 文本 | 如 http://127.0.0.1:7890 |
| HTTPS 代理 | 文本 | 同上 |
| 绕过代理 | 文本 | 逗号分隔域名列表 |

### Tab 4: 关于
- 版本号（自动读取 `__APP_VERSION__`）
- Git 版本（`git describe --tags`）
- 技术栈：Electron + React 19 + Vite 6
- 开源协议
- 更新日志链接

## 迁移步骤

### 第一步：不改现有功能，只加入口（当前迭代可做）
- 标题栏加齿轮图标
- 弹出「设置」模态框
- 只做「关于」Tab，放版本号

### 第二步：迁移环境管理（后续迭代）
- 环境管理从底部齿轮移到设置的「环境」Tab
- 底部齿轮改为快速切换选择器+跳转到设置

### 第三步：完整设置（后续迭代）
- 通用设置（语言/主题/超时）
- AI 设置
- 代理设置
- 导入/导出入口也挪到设置里
